### How to start?

1. Copy `.env.example` and make your own `.env`
2. Get your token from [@BotFather](https://telegram.me/botfather)
3. Update the `TOKEN` field in your `.env`
4. Run `pip install -r requirements` to install required modules
5. Run `make start`
